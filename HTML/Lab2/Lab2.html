1. ค้นหาข้อมูลว่าแท็ก <!doctype> มีหน้าที่อะไร?

Answer : มีหน้าที่ทำให้เว็บเบราเซอร์รู้ว่าโค้ดของไฟล์นี้ทั้งหมดเป็น html เวอร์ชั่น 5

2. ถ้าอยากให้มี tooltip ใน tag นั้นต้องใช้ attribute ใด?

Answer : ใช้ attribute title

3. แท็ก <b>, <i> ต่างจาก <strong>, <em> หรือไม่ อย่างไร?

Answer : ผลที่แสดงไม่ต่างกัน แต่จะให้ความสำคัญกับ tag strong มากกว่า tag b และให้ความสำคัญ tag em มากกว่า tag i

4. หา tag ที่สามารถแสดงข้อความกลับหลังได้

Answer : ใช้ tag bdo และ value = dir="rtl" ข้อความจะเรียงจากขวามาซ้าย
